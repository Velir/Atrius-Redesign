-------------- Jabberwocky.Extras.Polly.Sc --------------
---------------------------------------------------------

---- Installation ----

Be sure to install the included "sitecore packages\Jabberwocky Extras Polly Sc" sitecore package into your local solution.

This will add a new 'Error Handling' field section to the 'Rendering Options' template here: /sitecore/templates/layout/sections/rendering options/

Be sure to publish this path after the package has been installed.
